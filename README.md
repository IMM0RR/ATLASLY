# ATLASLY

Интерактивный трекер путешествий — веб-приложение для отметки посещённых стран и составления вишлиста мест, которые хочется посетить.

---
## О проекте

ATLASLY — фронтенд-приложение без бэкенда и базы данных. Все данные пользователя хранятся локально в cookies браузера, что означает полную работу без регистрации, авторизации и серверных запросов. Данные сохраняются между сессиями и доступны до тех пор, пока не очищены cookies браузера.

Приложение позволяет:

- Отмечать страны на интерактивной карте мира в двух режимах — "been" для посещённых стран и "wish" для желаемых
- Искать страны по названию через поиск с подсветкой на карте
- Масштабировать карту колёсиком мыши или слайдером, перемещаться по ней зажатой кнопкой мыши
- Просматривать профиль со статистикой — количество посещённых стран, процент от всего мира, вишлист
- Оценивать посещённые страны по пятибалльной шкале и сортировать список по алфавиту или рейтингу
- Получать достижения за количество посещённых стран и прокачивать уровень путешественника
- Смотреть факты о странах мира в профиле

## Стек

- React 18
- SCSS
- Vite — сборка и dev-сервер
- js-cookie — работа с cookies для хранения данных

---

## Хранение данных

Приложение не использует бэкенд. Все данные хранятся в cookies браузера:

- `atlasly_been` — список кодов посещённых стран
- `atlasly_wish` — список стран из вишлиста
- `atlasly_ratings` — оценки посещённых стран
- `atlasly_names` — названия стран для отображения в профиле
- `atlasly_claimed_ach` — полученные достижения (хранятся в localStorage)

Срок хранения cookies — 365 дней.

## Запуск проекта

Требования: Node.js версии 18 и выше.

Клонировать репозиторий:
```bash
git clone <url>
cd ATLASLY
```

Установить зависимости:
```bash
npm install
```

Запустить в режиме разработки:
```bash
npm run dev
```

Приложение откроется по адресу `http://localhost:5173`.

Сборка для продакшена:
```bash
npm run build
```

Предпросмотр собранного проекта:
```bash
npm run preview
```
